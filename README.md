# ⚽ Penalty Shootout Challenge

A football-themed carnival game developed using HTML, CSS, and JavaScript. Players test their penalty-taking skills against an AI goalkeeper or compete directly against another player in a local two-player duel.

---

## 🎮 Features

### Single Player Mode

* Player vs AI goalkeeper
* Three shooting directions:

  * Left
  * Center
  * Right
* Difficulty levels:

  * Easy
  * Medium
  * Hard
* 5-second countdown timer
* Goal and save detection
* Performance rating system
* Local leaderboard support

### Two Player Duel Mode

* Player A vs Player B
* Secret selection system
* Pass-device gameplay
* Role switching between Shooter and Goalkeeper
* Multiple penalty attempts per player
* Automatic winner determination
* Draw detection

### Gameplay Elements

* Football field interface
* Goalkeeper movement
* Ball movement animations
* Score tracking
* Timer system
* Match results screen
* Restart functionality

## 🕹️ How to Play

### Single Player

1. Enter your name.
2. Select difficulty.
3. Click **Start Match**.
4. Choose a shooting direction before the timer reaches zero.
5. Try to score as many goals as possible from 5 attempts.
6. View your final score and rating.

### Two Player Duel

1. Enter Player A and Player B names.
2. Select **Two Player Duel**.
3. Start the match.
4. The Shooter selects a direction.
5. Pass the device to the Goalkeeper.
6. The Goalkeeper selects a direction.
7. The result is revealed.
8. Players switch roles after their attempts.
9. The player with the highest score wins.
